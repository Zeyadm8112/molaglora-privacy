Privacy Policy

This privacy policy applies to the Molaglora application (hereby referred to as "Application") for mobile devices, created by Zeyad Mohamed (hereby referred to as "Service Provider") as a free service. This service is intended for use "AS IS".

1. Information Collection and Use

The Application collects the following user-provided information:

  • Full name
  • Email address
  • Password (securely stored in hashed form and not directly accessible)
  • Phone number
  • Age
  • Gender

The phone number may be verified using a one-time password (OTP) during registration to ensure account authenticity.

The Application may also collect limited technical information automatically, such as:

  • Device type and operating system
  • IP address (for security and fraud prevention purposes)

2. How Information is Used

The Service Provider uses collected information strictly for the following purposes:

  • Creating and managing user accounts
  • Enabling login and authentication
  • Verifying phone number ownership during registration
  • Managing appointment bookings for beauty centers and dentistry services
  • Ensuring platform security and preventing fraud
  • Improving application performance and stability

3. Account Registration and Authentication

Users register and log in using email and password.

Passwords are securely stored using hashing techniques and are never stored in plain text.

During registration, users may be required to verify their phone number using an OTP code sent to the provided number. This step is used only for verification purposes and is not required for future logins.

4. Data Storage

User data is securely stored on a custom backend server managed by the Service Provider. Data is not stored locally on the device except for temporary session-related information.

5. Third-Party Services

The Application may use trusted third-party services strictly for operational purposes, such as:

  • OTP delivery services for phone number verification
  • Hosting and backend infrastructure providers

These providers process data only to perform their required functions and are not permitted to use it for any other purpose.

6. Data Sharing

The Service Provider does not sell, rent, or share user personal data with third parties.

User data may only be disclosed in the following cases:

  • When required by law or legal process
  • When necessary to protect user safety, prevent fraud, or enforce security
  • When required for core service functionality (e.g., OTP delivery systems)

7. Data Retention and Deletion

Users can request deletion of their personal account information within the Application.

Upon deletion:

  • Personal account data (name, email, phone number, credentials) will be removed from the system
  • Certain non-personal operational records such as booking history or transaction receipts may be retained for business, legal, or financial compliance purposes

These retained records do not directly identify users outside the system context.

8. Location Data

The Application does not collect or use precise or real-time location data.

9. Children’s Privacy

The Application is intended for users aged 16 and above.

The Service Provider does not knowingly collect personal information from users under the age of 16. If such data is discovered, it will be deleted promptly.

10. Security

The Service Provider implements appropriate technical and organizational measures to protect user data, including:

  • HTTPS encrypted communication
  • Secure authentication mechanisms
  • Hashed password storage
  • Protected backend infrastructure

However, no system can guarantee absolute security.

11. Marketing and Advertising

The Application does not use user data for marketing or advertising purposes.
